ROS2 WORKSPACE FLOW (har lab ka base)

• mkdir -p ~/ros2_ws/src
• cd ~/ros2_ws
• colcon build
• source install/setup.bash

PACKAGE CREATE

• cd ~/ros2_ws/src
• ros2 pkg create --build-type ament_python package_name

PYTHON NODE SETUP

• cd package_name/package_name
• touch node.py
• chmod +x node.py

SETUP.PY ENTRY POINT (bohat important)

entry_points={
'console_scripts': [
'node_name = package_name.node:main',
],
}

• cd ~/ros2_ws

NODE RUN

• ros2 run package_name node_name
